### SSH主要用于远程登录

> SSH的默认端口是22
> 公钥被接受以后，就会被保存 $HOME/.ssh/known_hosts 之中
> 系统保存对所有用户可信赖的远程主机的公钥在 /etc/ssh/ssh_known_hosts 之中

```
$ ssh user@host (-p port)
```

### 重启远程主机的ssh服务

```
$ service ssh restart // ubuntu系统
$ /etc/init.d/ssh restart // debian系统
```
