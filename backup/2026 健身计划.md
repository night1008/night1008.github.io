# 一、总体目标

**阶段目标（建议）**

| 时间    | 目标         |
| ----- | ---------- |
| 第1个月  | 体重 +2kg    |
| 第3个月  | 体重 +5kg    |
| 第6个月  | 体重 +8~10kg |
| 第12个月 | 体重 65kg 左右 |

重点原则：

1️⃣ **热量盈余**（每天多吃300-500 kcal）
2️⃣ **力量训练为主**
3️⃣ **渐进超负荷**（重量慢慢增加）
4️⃣ **保证蛋白质摄入**

# 二、每周训练计划（建议一周4练）

新手不要天天练，否则恢复不了。

**训练安排**

| 星期 | 训练       |
| -- | -------- |
| 周一 | 胸 + 三头   |
| 周二 | 背 + 二头   |
| 周三 | 休息       |
| 周四 | 腿 + 核心   |
| 周五 | 肩        |
| 周六 | 轻运动 / 拉伸 |
| 周日 | 休息       |

每次训练 **60-75分钟**。

---

# 三、具体训练动作

## 周一：胸 + 三头

1. 杠铃卧推
   4组 × 8-10次

2. 哑铃卧推
   4组 × 10次

3. 上斜哑铃卧推
   3组 × 10次

4. 双杠臂屈伸
   3组 × 8-10次

5. 绳索下压（三头）
   4组 × 12次

6. 哑铃臂屈伸
   3组 × 12次

---

## 周二：背 + 二头

1. 引体向上
   4组 × 尽量多

2. 高位下拉
   4组 × 10次

3. 杠铃划船
   4组 × 10次

4. 坐姿划船
   3组 × 12次

5. 杠铃弯举
   4组 × 10次

6. 哑铃弯举
   3组 × 12次

---

## 周四：腿 + 核心

1. 深蹲
   4组 × 8-10次

2. 腿举机
   4组 × 10次

3. 罗马尼亚硬拉
   4组 × 10次

4. 腿弯举
   3组 × 12次

5. 卷腹
   4组 × 15次

6. 平板支撑
   3组 × 60秒

腿部训练非常重要，因为 **腿部训练会促进整体增肌和睾酮分泌**。

---

## 周五：肩

1. 杠铃推举
   4组 × 8-10次

2. 哑铃侧平举
   4组 × 12次

3. 俯身侧平举（后束）
   4组 × 12次

4. 哑铃前平举
   3组 × 12次

5. 耸肩
   4组 × 12次

---

# 四、重量选择原则

选择重量时遵循：

最后 **2次比较吃力，但能完成**

例如：

8-10次
第9次开始吃力 = 正确重量

每周：

重量增加 **2.5kg-5kg**

这叫 **渐进超负荷**。

# 七、睡眠和恢复

睡眠对增肌非常重要。

保证：

**每天7-8小时**

建议：

23:00前睡觉

---

# 八、新手最容易犯的错误

❌ 训练太多
❌ 吃得不够
❌ 不练腿
❌ 重量不增加

增肌核心公式：

**训练刺激 + 热量盈余 + 睡眠**
